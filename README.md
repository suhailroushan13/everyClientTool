# everyClientTool

## Client Tools
- Axios
- Node-Fetch
- Python Request
- Curl
- Wget
- PostMan
- Insomania

## For Ubuntu Machines

### Open Config Folder and Add URL

```bash
sudo apt install curl
sudo apt-get install wget
git clone git@github.com:suhailroushan13/everyClientTool.git
cd everyClientTool
npm install
npm start
```
